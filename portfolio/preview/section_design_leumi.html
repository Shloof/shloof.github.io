Design

Search city query
The user clicks the search field, starts typing "אשדוד," and selects it from the autocomplete suggestions that appear in the dropdown.
Search city Results
The system returns search results with a list of branch managers in אשדוד. The user selects a branch manager's card, which opens additional information.
WhatsApp redirection
The user clicks the WhatsApp button to chat with the branch manager. It's available on both the closed and open card.
Search Branch number query
The user clicks on the search field and types their branch number. The system returns one result - the branch manager for the number entered.
Search error & suggestions
The user enters a misspelled query and gets no results, but input assistance offers helpful suggestions.
Commitment statement
The user scrolls down to the branch managers' statement, begins reading, and clicks on "להצהרה המלאה" (Read the full statement). After finishing, he close the statement.
Desktop
I designed this project mobile-first, then optimized the desktop layout to use screen space efficiently by showing manager details alongside extra info without clutter.
